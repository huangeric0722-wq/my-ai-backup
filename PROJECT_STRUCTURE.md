# 賈維斯工作區架構說明 (Workspace Structure)

## 🧠 MEMORY (長期記憶與家規)
存放 Eric 的個人偏好、API 憑證、以及公司核心業務背景。
- `MEMORY.md`: 核心規則與目前狀態。
- `memory/`: 存放在長期穩定的資產（如 API Token）。

## 📂 PROJECTS (研究與開發專區)
存放所有「正在進行」或「已開發完成」的專案。

### 1. `projects/research/` (研究議題)
存放所有技術研究報告、簡報與 Phase 學習筆記。
- 例如: `OBSERVABILITY_PHASE3_REPORT.md`

### 2. `projects/tools/` (開發好的專案/工具)
存放寫好的腳本、工具程式。
- 例如: PowerShell 腳本、Python 自動化工具。

### 3. `projects/uipath/` (RPA 學習專案)
專門存放 UiPath 相關的學習進度與檢查清單。
- 例如: `UIPATH_SECURITY_CHECKLIST.md`
